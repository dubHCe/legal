# Central Alerts — Legal

Hosts the privacy policy and other legal documents for the Central Alerts
mobile application, served at **<https://legal.bestcontrader.com>** via
GitHub Pages.

## Files in this repo

| File | Purpose |
|------|---------|
| `privacy.md` | Privacy policy (published at `/privacy/`) |
| `index.md` | Landing page at the site root |
| `_config.yml` | Jekyll configuration (title, theme, plugins) |
| `assets/css/style.scss` | Navy-branded styling overrides |
| `Gemfile` | Ruby dependencies for local preview (optional) |
| `CNAME` | Custom domain configuration |

## Editing the policy

1. Edit `privacy.md` with your changes.
2. Bump the "Effective date" at the top and the "Last updated" at the bottom.
3. Commit and push to `main`. GitHub Pages rebuilds and publishes within
   about a minute.
